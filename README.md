## ***Installation*** ⚙️

*To install the service you should follow these steps:*

1️⃣ *Run your command console.*

2️⃣ *Execute the following command*

```bash
npm install
```

## ***Local Deploy*** 📦
   
*To run the script you should follow these steps:*

1️⃣ *Run your command console.*

2️⃣ *Execute the following command*

```bash
npm run dev
```
